# Banl-Customer-Segementation
# 🏦 Bank Customer Segmentation Dashboard

## 📌 Tools Used
- Power BI
- Excel
- Power Query
- DAX

## 📊 Project Overview
This project analyzes bank customer data to segment customers based on income, risk, demographics, and transaction behavior.

## 📁 Features
- Data cleaning using Power Query
- Customer segmentation by income & credit score
- Risk analysis (High, Medium, Low)
- Demographic insights (Region, Gender)
- KPI dashboards for business decisions

## 📈 Key Insights
- High-income customers are key targets for premium banking
- South region has higher-risk customers
- Credit score is a major segmentation factor

## 🚀 Outcome
Improved understanding of customer behavior for targeted marketing and retention strategies.
